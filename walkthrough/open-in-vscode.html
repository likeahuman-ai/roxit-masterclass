Wat gebeurt er als je de launcher start?
1

Dubbelklik Roxit.command (Mac) of Roxit.bat (Windows)

2

Launcher maakt ~/Desktop/roxit-workshop aan en vult het met de starterbestanden auto

3

VS Code opent de workspace auto

4

Docker container start met dezelfde map als /workspace auto

Twee vensters, een project

Hoe je werkt
A

VS Code - bestanden bekijken en bewerken. Hier schrijf je code, bekijk je de structuur, en werk je met je editor tooling.

B

Terminal (Roxit venster) - commando's uitvoeren. Hier draai je claude, pnpm dev, vercel deploy, etc.

Belangrijk

De VS Code terminal opent op je laptop, niet in de container. claude en pnpm werken daar niet. Gebruik altijd het Roxit terminal-venster voor commando's.

VS Code niet geinstalleerd?

Geen probleem - de launcher opent dan Finder (Mac) of Explorer (Windows). Je kunt bestanden gewoon bekijken en bewerken met elke editor. De Roxit terminal is waar je claude draait.

VS Code opende niet automatisch?

Handmatig openen
1

Open VS Code

2

File > Open Folder

3

Navigeer naar ~/Desktop/roxit-workshop (Mac) of C:\Users\<naam>\Desktop\roxit-workshop (Windows)
